dao_treasury.sorting.rules.ignore.shitcoins module

This module is used to ignore any transactions that involve shitcoins in the SHITCOINS mapping.

dao_treasury.sorting.rules.ignore.shitcoins.is_shitcoin(tx)[source]

This category is for any token transfers involving any shitcoin which was added to your database before the shitcoin was added to eth_portfolio.SHITCOINS.

Since the tx would now be excluded from the db entirely if you did a clean pull, we can safely ignore it.
